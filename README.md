# Job Jugaad AI 🚀

Job Jugaad AI is an intelligent platform designed to help candidates prepare for their dream jobs. It offers AI-driven resume analysis against job descriptions, mock interview practice, and a personalized dashboard to track your progress.

## ✨ Features

- **AI Resume Analysis**: Upload your resume and a job description to get a detailed fit analysis and actionable feedback.
- **Mock Interviews**: Practice your interview skills with AI-generated questions tailored to your profile.
- **User Dashboard**: Track your past analyses, view history, and monitor your overall preparation stats.
- **Seamless Authentication**: Secure login using Google OAuth or standard email/password provided by Supabase.

## 🛠️ Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Radix UI components, Framer Motion for animations
- **Backend**: Node.js, Express, TypeScript
- **Database**: PostgreSQL (hosted on Supabase)
- **AI Integration**: Groq API for lightning-fast, intelligent analysis
